# Hashi Puzzle Solver (Prolog)

This project implements the first part of a **Hashi puzzle solver** using Prolog.

Hashi is a logic puzzle played on a grid where islands must be connected by bridges according to specific rules:
- Each island has a required number of bridges
- Bridges can be horizontal or vertical
- Bridges cannot cross
- At most two bridges connect any pair of islands
- All islands must be connected

## Features
- Representation of puzzles as grids
- Extraction and manipulation of islands
- Detection of neighboring islands
- State management during puzzle solving
- Bridge creation and validation
- Handling of completed islands

## Technologies
- Prolog (SWI-Prolog recommended)
